# Cómo correr el proyecto
1. Instalar dependencias en carpeta raíz
```
npm install
```
2. Instalar dependencias en carpeta server
```
cd server
npm install
cd ..
```
3. Levantar el server node con express en localhost:8000
```
npm run server
```
4. Levantar el proyecto en React en localhost:3000
```
npm run start
```
Automáticamente se levantará una nueva pestaña en su navegador con el proyecto funcionando
